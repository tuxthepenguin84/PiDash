# PiDash

PiDash is custom dashboard used to show weather, stocks, transportation info, webcams, calendars, and more.

![PiDash](pidash.png?raw=true "PiDash")

## Getting Started

Uses HTML 5 & Python

### Prerequisites

Python

### Installing

Built using HTML 5 and Python 3. To update index.html run:
```
python3 main.py
```
